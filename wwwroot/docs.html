How to use the API

The setup for the Characters that you will implement needs to have the setup that you see below. You cannot add a string for image unfortunately that will be implemented later on in the API.

ID: has a auto increment so you dont need to worry about that.
Name: is a string and can be null if you dont have a name for the character. However if you dont input Name you will neither be able to have an image to the character.
Game: is a string and can be null if you dont have a game for the character. 

                            
"ICharacter interface"
{
    "id": number,
    "name": string,
    "game": string 
    "image": string, //casting name image
}

You have a few ways to use the API HTTP calls.
The existing ones are ways to gather the information from the database.
Add information to the database.
Delete information from the database.
Alter some of the information in the database.
Collect information from the database based on the ID.
Insert information to the database while adding a image with the character.
You can also get information about a character by the name

"HTTP Calls"
{
    "Get": "https://localhost:****/api/characters",

    "GetById": "https://localhost:****/api/characters/GetById/{id}",
    "PostImage": "https://localhost:****/api/characters/uploadImage/{name}",
    "getByName": "https://localhost:****/api/characters/GetByName/{name}"

}
